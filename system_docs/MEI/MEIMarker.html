MGD EDITING INTERFACE: Design and Requirements - Markers


Introduction

The Markers constitute one of the backbones of the MGD database. Virtually no data is entered into the MGD database without a Marker of some kind associated with it.

Markers make their way into MGD via the Nomenclature process. Novel and Reserved Markers are broadcast from the Nomenclature database to MGD by Nomen editors via a broadcast process. For more information regarding the Nomenclature database and the Broadcast process, refer to:

  • Nomenclature Requirements

Deletions

A Marker cannot be deleted if it is referenced by a:
  1. Experiment
  2. Molecular Probe & Segment
  3. Homology
  4. MLC entry
  5. GO Annotation
  6. Expression Assay
  7. Another Marker's History
  8. Another Marker's Current

Modifying Chromosome Values

When modifications are made to the Chromosome value of the Marker, certain rules apply:
  1. If the Chromosome is being changed to UN, then set the Offset to -999.0.
  2. If the Chromosome was known and is changed to another known, then set the MGD and CC Offsets to -1.0. However, if the Marker is an Anchor Locus, then the Anchor record must be deleted before the Chromosome value can be modified.

Other Names/References

The Other Names/Reference table can be used to edit:
  1. Other Names which do not have References
    • Enter Other Name only. Modify record.
  2. Other Names which do have References
    • Enter Other Name and J:. Modify record.
  3. References only (additional references, MLC indexing, etc.)
    • Enter J: only. Modify record.
References which are new or are added to existing other names will become immediately visible on the Marker Reference display in the Web interface (WI). Deletions (removals) of References may not be reflected in the WI until the next day.

This table can also be used to modify the Review Status of a Reference. To do so:

  1. Search for a marker record.
  2. Select the row of the Reference you wish to modify.
  3. Use the Review? pulldown menu in the side panel to select the review value you wish to assign to the Reference.
  4. Modify the record.

Miscellaneous

  1. The Add function has been disabled on this form. All new marker symbols must be input into MGD using the Nomenclature form.

  2. Allele symbols and names may be searched using the Marker form, but can only be edited using the Allele form.

  3. The Withdrawal process will modify the Status to withdrawn.

  4. Editing of the # column is implemented for the following tables:
    • Marker History

  5. Editing of certain types of accession numbers added during the electronic data loads is restricted. These electronically loaded accession numbers are distinguished by load references. If the user attempts to edit the Accession numbers of a marker record from a data load, the following error message will appear: 
    	Accession ID is referenced by J:#####, "some" Database Download.
	Cannot edit.
