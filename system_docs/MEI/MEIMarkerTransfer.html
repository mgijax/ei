MGI: Marker Transfer Report

Introduction

The Marker Transfer Report tells the user which datasets are involved in a transfer from one Marker Symbol/J: pair to another symbol. This process does not actually transfer any data.

Initiating the Marker Transfer Report

A Marker Transfer Report is initiated from the MGD Editorial Interface's Marker form using the Utilities->Marker Transfer Report menu item in the top menu bar.

The user is required to enter:

  1. Old Symbol
  2. New Symbol
  3. J:
This data is validated before processing can continue.
  1. The user enters the Old Symbol and TAB.

  2. The user enters the New Symbol and TAB.

  3. The user enters the J: and TAB.

  4. If there is no data to transfer between the 2 symbols, the message: 
    	There is no data to transfer between these 2 symbols
    is displayed.

  5. If there is data to transfer, the toggle buttons will indicate which datasets contain data to be transferred.

  6. The user clicks the Done button to return to the main Marker form.
