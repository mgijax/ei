Introduction

MGD stores Marker information and Accession numbers for non-mouse Markers which are used in the Homology portion of the database.

From the Web Homology detail page, MGD links to species-specific databases (SSDBs) either by Marker symbol or by SSDB Accession number. Not every SSDB provides Accession numbers for their Marker objects.

SSDBs are also used to maintain MGD nomenclature and mapping information for non-mouse markers.

The Marker information and Accession numbers for the non-mouse Markers must be updated on a regular basis to reflect the SSDB changes in nomenclature, mapping position, etc.

Getting the Data Dumps

The species-specific Accession numbers are loaded into MGD by an automated process which uses data dumps provided by the SSDBs.

RATMAP

We are currently using RATMAP to update MGD rat mapping and accession information. We are not updating MGD Rat symbols from RATMAP.
  1. All MGD Rat symbols which do not contain a RATMAP ID are updated with the correct RATMAP ID based on a symbol match between MGD and the RATMAP symbol.

  2. All MGD Rat symbols are updated from the RATMAP file based on a RATMAP ID match between MGD and RATMAP. The fields which are updated are:
    • chromosome
    • offset

  3. RATMAP Data & QC Reports

  4. In addition, we retrieve the following files from the RATMAP ftp site:
    • RATMAP Homolog file (RATMAP.Homolog.txt)

  5. Executed from a sybase cron on titan: 
    	/usr/local/mgi/dataload/ratmapload/RATMAPupdate.sh -l
