Introduction

This document describes the procedures for entering records into the Nomenclature datatabase.

Entering a New Symbol

  1. Clear the form.
  2. Enter values using guidelines below.
  3. Click Add.

Entering a Withdrawn Symbol

  1. Query for the original, approved Symbol.
  2. Modify the values using the guidelines below.
  3. Click Modify.
  4. If the new symbol(s) do not exist in MGD, then enter them into Nomen by following the Entering a New Symbol procedures above.
