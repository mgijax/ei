MGD EDITING INTERFACE: Design and Requirements - Mapping


Introduction

Mapping allows entry of the following Experiment Types:
  • Cross
  • Recombinant Inbred (RI)
  • Somatic Cell Hybrid (Hybrid)
  • InSitu Hybridization (InSitu)
  • FISH
  • Text
  • Physical Mapping (Physical)

An Experiment master record contains a Reference and a list of Primary Markers. This Primary Marker list is not viewed on the Web and may prove to be obsolete. In any case, it is useful, especially for propagating Marker symbol changes to all Experiments in a given Reference.

Multiple instances of any Experiment Type may be defined for a given Reference. Each Experiment of a given Type for a Reference is assigned a sequential tag number by the system.

Notes may be attached to the entire Reference or to each individual Experiment.

  8. Cross Experiments
    • Use the View->Cross Listing dialog to select a valid, non-anonymous Cross. This listing is generated from the Master Cross table.

      A Cross selection will populate the following fields:

      • Cross
      • Cross Type
      • Female Strain
      • Male Strain
      • Allele/Strain 1
      • Allele/Strain 2
      • Offspring type indicates alleles from segregating parent
      • F1 direction known

      A Cross selection will also construct the Female and Male genotypes by counting the number of Markers in the Experiment Marker table which have Matrix? equal to yes. If the Experiment Chromosome is X, then the right side of the Male genotype will default to /Y.

      If a valid, non-anonymous Cross is not selected, then the user must manually fill in the fields which define the Cross.

    • The number of Haplotypes should equal the number of Markers in the Experiment Marker table which have Matrix? equal to yes. Modifications allowed on invalid entries.

  9. RI Experiments
    • Use the View->RI Listing dialog to select a valid RI/RC set. This listing is generated from the Master RI Set table.

      A RI selection will populate the following fields:

      • Designation
      • Origin
      • Abbrev1
      • Abbrev2

    • Valid alleles for the Haplotypes table listed below. Modifications denied on invalid entries.
      • Abbrev1
      • Abbrev2
      • . (untyped)
      • Z (heterozygous)
      • X (new allele)
      • ? (ambiguous typing)
      • * (inconsistent typing)

    • The number of Haplotypes must equal the number of Animals for the RI Set. Modifications denied on invalid entries.

Defaults

  1. When the Experiment Type is FISH and a Marker is selected from the Primary Marker selection list, the Assay Type for that Marker will default to FISH.

  2. When the Experiment Type is InSitu and a Marker is selected from the Primary Marker selection list, the Assay Type for that Marker will default to in situ hybridization.

Deletions

There are no restrictions on deleting an Experiment or all Experiments for a given Reference.

Miscellaneous

  1. To view/edit Experiment master records, set the Experiments toggle OFF.

  2. To view/edit Experiment master records, set the Experiments toggle ON.

  3. If a Primary Marker for a Reference is modified, then that modification will propagate to all of the Experiments for that Reference.

    For example, if symbol Ren1 is changed to symbol Ren2 for J:1111, then all instances of Ren1 in all Experiments for J:1111 will be changed to Ren2.

  4. The deletion of a master Experiment record will delete the Experiment master and all its Experiments.

  5. If the Chromosome value of an Experiment Marker is reserved or unknown, then its Chromosome value in the master Marker table will be updated to the Experiment Chromosome value when the record is modified.
