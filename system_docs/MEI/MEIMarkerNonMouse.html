MGD EDITING INTERFACE: Design and Requirements - Non-Mouse Markers


Introduction

Non-mouse markers are stored in the database as part of the Homology data. These markers have traditionally been edited via the Homology form. This form also allows users to edit non-mouse markers but it has the added capability of allowing users to attach Sequence Accession Numbers (with References) to non-mouse markers.
