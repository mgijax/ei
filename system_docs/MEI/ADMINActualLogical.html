ADMINISTRATION EDITING INTERFACE: Design and Requirements - Actual/Logical DB

Introduction

The Logical and Actual DB tables are used to define logical data resources and actual URL links to these resources so that HTML links can be automatically generated for objects attached to the logical data resources.

For example, a logical data resource Sequence DB is defined as a nucleotide data resource. There are 4 distinct databases which provide URLs for establishing HTML links to this data resource using the same Accession numbers:

  1. DDBJ (Japan)
  2. EMBL (UK)
  3. GenBank (US)
  4. GSDB (US)
Sequence accession numbers are attached to this logical data resource via the Editorial interface. When the corresponding data object is displayed in the Web interface, a link will be established to each of the 4 actual databases described above from the Sequence accession numbers defined. See MGI:21151.
