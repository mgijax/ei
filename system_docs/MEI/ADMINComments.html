ADMINISTRATION EDITING INTERFACE: Design and Requirements - Table Comments

Introduction

The table and column comments are used to provide schema documentation for software engineers and other technical users of the database. Some comments may include biological definitions, but these are limited in scope. Full biological documentation is available for GXD and will be available for MGD in the future.

Table and column comments can also be viewed by using the MGI Schema Browser.

Miscellaneous

  1. All inserts and deletions to the Tables and Columns tables are performed by stored procedures and triggers which are run nightly. This is to insure these table records are synchronized with the system sysobjects and syscomments tables.

  2. Only modfications to Table Description, Column Description and Column Example are allowed via the editing form.
