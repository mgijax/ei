GXD EDITING INTERFACE: Design and Requirements - Index/Stages

Introduction

The GXD Index is a precursor to the full GXD suite of tools which enables the users to cross reference an expression reference with the gene of interest, the assays used to detect expression of the gene, and the days post conception (DPC) of the animals used in the assay.

This resource provides a simple way for the Editors to index the expression references which will eventually be fully coded for entry into the Antigen/Antibody/Assay sections of the GXD database.

Deletions

There are no restrictions on deleting a GXD Index record.

Miscellaneous

  1. To select a stage and days post conception (DPC), use the mouse to click on the appropriate cell. An X will appear in the cell and will begin the flash. The flash is simply a visual aid to draw the user's attention to those cells which have been selected.

  2. After the user clicks Add or Modify, if the GXD Index record does not have any Stage/DPC information selected, then the warning message: 
    	No Stages have been selected for this record.
    will be displayed.

  3. To add a new Assay type to the GXD Index:
    • Add the new Assay type to the "GXD Assay Type" vocabulary in the Simple Vocabulary module.
    • Re-launch the GXD Index module.

  4. The Priority value for a given Index record is propagated to all Index records that use the same J Number.

  5. The Coded? value is determined by searching for the J Number in the GXD Assay module. If the J Number is found in the GXD Assay module, then Coded? = Yes else No.
