Introduction

The Controlled Vocab editing form enables editing of many controlled vocabulary lists in the database.

Some controlled vocabulary correspond to Option Menus:

  • Marker Types
  • Reference Review Status
  • Molecular Segment Vector Types
  • Antibody Types
  • Antibody Classes
  • Assay Types
  • Expression Patterns
  • Expression Strengths
  • Gel RNA Types
  • Gel Units
  • Gel Lane Control
  • Image Field Types
  • Antibody/Probe Prep Labels
  • Probe Prep Label Coverages
  • Probe Prep Sense Values
  • Probe Prep Visualized Methods
  • Secondary Antibody Preps
  • Specimen Embedding Methods
  • Specimen Fixation Methods
  • Marker Event Reasons
  • Allele Types
  • Inheritance Modes
  • Molecular Mutations

Some controlled vocabulary correspond to Lists:

  • Homology Assays
  • Mapping Assays
  • MLC Classes

Fields

As the user selects Table from the option menu, the bottom half of the Controlled Vocab form may change depending on the Table selected and different fields will become available for viewing/editing. The # Records will also change to reflect the correct row count for the selected Table.
  • Required

    2. Name

    2. Gel Assay

  • Required for Homology Assays
    1. Assay Abbreviation (for Web display)

Defaults

None

Deletions

A controlled vocabulary cannot be deleted if it is referenced by an object which cross-references it elsewhere in the database. The following controlled vocabulary can never be modified or deleted:
  1. Not Specified
  2. Not Applicable

Miscellaneous

  1. To see edits in the corresponding option menu after a controlled vocabulary table edit, exit and re-enter the form which uses the applicable option menu.

    For example, if an edit is made to Vector Types and the Molecular Probes & Segments form is currently instantiated, exit the form and re-enter it to view the current edits.

  2. To see edits in the corresponding list after a controlled vocabulary table edit, click on the label in the applicable list.

    For example, if an edit is made to Homology Assays and the Homology form is currently instantiated, click on the ## Homology Assays label to re-load the list. The current edit will be reflected in the new list.
