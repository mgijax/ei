MGI EDITING INTERFACE: Design and Requirements - Organism


Introduction

The master list of Organism contains the Latin and common names for over 75 mammalian organism. Data sets which cross reference to the master Organism list are:
  • Molecular Probes & Segment Libraries (Sources)
  • Orthology

The master Organism list also cross-references a Chromosome Order table. This table provides an ordered list of all chromosomes for a given organism. Ordering of chromosomes is essential for creating reports and constructing the Oxford Grid.

In addition, the Organism mouse, laboratory cross-references a list of Anchor loci. Anchor loci are loci that have been well-defined and well-mapped on their chromosome. These loci are used to construct the Web mini-maps on the Marker detail screen.

Deletions

A Organism cannot be deleted if it is referenced by a:
  1. Marker
  2. Molecular Probe & Segment
  3. Orthology

Miscellaneous

  1. The MGI Types for Which This Organism is Valid is used to define the organisms relevant for each MGI Type. The EI uses these definitions to build the Organsim pulldown option lists in the Antibody, Antigen, Marker, Orthology, Molecular Segments and Sequence EI modules.

  2. Anchor Loci are only relevant for mouse, laboratory.
