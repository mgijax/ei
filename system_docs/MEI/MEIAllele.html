Deletions

  1. Experiment
  2. GXD Genotype

Allele Merge

The Allele Merge process allows the user to merge two pre-existing Allele records into one Allele record.
  1. Click on Utilities->Allele Merge to bring up the Allele Merge dialog.
  2. Enter the Marker Symbol in the Marker dialog field. This field will be verified.
  3. Enter the Old Allele Symbol in the Old Allele Symbol dialog field. The Old Allele Symbol must currently exist as an Allele Symbol for the Marker Symbol.
  4. Enter the New Allele Symbol in the New Allele Symbol dialog field. The New Allele Symbol must currently exist as an Allele Symbol for the Marker Symbol.
  5. Click Process Merge to process the merge.
  6. Click Done to return to the Marker editing form.

Miscellaneous

  1. If the Allele Status is Approved, then the Marker Symbol must be an Approved symbol, else the Marker Symbol can be a NomenDB TaintReady, Reserved or Approved symbol.

  2. If the controlled vocabulary term Other (see notes) is selected from Allele Type or Inheritance Mode, then General Notes are required.

  3. If the controlled vocabulary term Other (see notes) is selected from Molecular Mutation, then Molecular Notes are required.
