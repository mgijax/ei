  • Molecular Segment Libraries (Source)

Deletions

A Strain cannot be deleted if it is referenced by a:
  1. Allele
  2. Molecular Segment Library (Source)
  3. Molecular Segment RFLP
  4. master Cross
  5. Cross Experiment
  6. InSitu Experiment
  7. FISH Experiment
  8. GXD Genotype

Merge

The Strain form supports three types of merges. All Strains involved in a merge must have the same JAX Mice Registry number. The target Strain will always end up with the JAX Mice Registry number.
  1. Merge Non-standard Strain into Standard Strain

    Merges all records tagged as Non-standard for the given Strain into the corresponding Standard record. If more than one Standard record is found, then the merge cannot be processed.

  2. Merge Standard Strains into Non-standard Strain

    Merges all records tagged as Standard for the given Strain into the corresponding Non-standard record. If more than one Non-standard record is found, then the merge cannot be processed.

  3. Merge old Strain into new Strain

    Merges two distinct Strain records into the New Strain and deletes the Old Strain. Propagates changes out to all cross-references to the Old Strain:

    1. Allele
    2. Molecular Segment Library (Source)
    3. Molecular Segment RFLP
    4. master Cross
    5. Cross Experiment
    6. InSitu Experiment
    7. FISH Experiment
    8. GXD Genotype

Retrieving Other Strain Information

  1. To retrieve all References which cross-reference the current Strain, click on Retrieve Info in the References table. This will list the J: Accession Number for each Mapping, Expression or RFLP record which uses the current Strain.
  2. To retrieve all Data Sets which cross-reference the current Strain, click on Retrieve Info in the Data Sets table. This will list the MGI: Accession Number for each Molecular Segment or GXD Antigen which uses the current Strain.
