MGD EDITING INTERFACE: Design and Requirements - Species


Introduction

The master list of Species contains the Latin and common names for over 75 mammalian species. Data sets which cross reference to the master Species list are:
  • Molecular Probes & Segment Libraries (Sources)
  • Homology

The master Species list also cross-references a Chromosome Order table. This table provides an ordered list of all chromosomes for a given species. Ordering of chromosomes is essential for creating reports and constructing the Oxford Grid.

In addition, the Species mouse, laboratory cross-references a list of Anchor loci. Anchor loci are loci that have been well-defined and well-mapped on their chromosome. These loci are used to construct the Web mini-maps on the Marker detail screen.

Deletions

A Species cannot be deleted if it is referenced by a:
  1. Marker
  2. Molecular Probe & Segment
  3. Homology

Miscellaneous

  1. Anchor Loci are only relevant for mouse, laboratory.
