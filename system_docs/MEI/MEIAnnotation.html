MGI EDITING INTERFACE: Design and Requirements - Vocabulary Annotation


Introduction

This module is used to annotate a specific vocabulary (GO, Mammalian Phenotype) to a specific type of database object (Marker, Genotype).

Editing Form

GIF Image

Data Verifications

  1. Term Acc ID

  2. J Numbers (see Data Verifications section on J Numbers)

  3. Evidence Code

Fields

  • Required
    1. Annotation Type
    2. MGI Accession ID/Object
    3. Term Acc ID
    4. J:
    5. Evidence

  • Not Required
    1. NOT
    2. Inferred From
    3. Modified By
    4. Date
    5. Notes

Defaults

  1. DAG is automatically filled in based on the Term Acc ID.

  2. Vocabulary Term is automatically filled in based on the Term Acc ID.

  3. Citation is automatically filled in based on the J:.

  4. Modified By is automatically filled in based on user.
