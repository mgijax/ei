Introduction

The Simple Vocab editing form enables editing of many simple vocabulary lists in the database.

Examples:

  • Curation State
  • GO Evidence Codes
  • GXD Index Assay
  • GXD Index Priority
  • GXD Index Stages
  • GXD Reporter Gene
  • InterPro Domains
  • Nomen Status
  • PhenoSlim
  • PhenoSlim Evidence Codes
  • Segment Type
  • Super Standard Strain

Deletions

A simple vocabulary cannot be deleted if it is referenced by an object which cross-references it elsewhere in the database.

Miscellaneous

  1. To see edits in the corresponding option menu after a simple vocabulary table edit, exit and re-enter the form which uses the applicable option menu.

    For example, if an edit is made to GXD Index Priority and the Index form is currently instantiated, exit the form and re-enter it to view the current edits.

  2. To see edits in the corresponding list after a simple vocabulary table edit, click on the label in the applicable list.

    For example, if an edit is made to GO Evidence Codes and the Vocabulary Annotation form is currently instantiated, click on the ## Evidence Codes label to re-load the list. The current edit will be reflected in the new list.
