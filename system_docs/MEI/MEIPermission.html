Assume user needs insert, update and delete permissions for database tables listed.
Editing ScopeDatabase TablesNotes
References BIB_*, ACC_Accession*
  1. If user only needs to edit Data Set info, then BIB_Refs only
Markers MRK_*, ACC_Accession*  
Homology HMD_*, MRK_*, ACC_Accession*
  1. Users need to add non-mouse species to Marker table

  1. insert only on strains tables
Mapping MLD_*, CRS_Cross, RI_RISet, MRK_Marker, ACC_Accession*, strains..MLP_Strain, strains..MLP_Notes
  1. Trigger may update MRK_Marker.chromosome.
  2. Insert only on strains tables.
Mapping Panels MLD_*, CRS_*, MRK_Marker, ACC_Accession*
  1. These tables are loaded by a cron data load program.
RI Summary RI_Summary*
  1. These tables are loaded by a cron data load program RI_Summary.sql.

  1. Antigen entry requires PRB_Source;Trigger may insert records into PRB_Reference.
  2. Assay Genotype dialog may insert records into ALL_Allele.
  3. Insert only on strains tables.
