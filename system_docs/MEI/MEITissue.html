Deletions

A Tissue cannot be deleted if it is referenced by a:
  1. Molecular Segment Library (Source)

Merge

The Tissue form supports three types of merges:
  1. Merge Non-standard Tissue into Standard Tissue

    Merges all records tagged as Non-standard for the given Tissue into the corresponding Standard record. If more than one Standard record is found, then the merge cannot be processed.

  2. Merge Standard Tissues into Non-standard Tissue

    Merges all records tagged as Standard for the given Tissue into the corresponding Non-standard record. If more than one Non-standard record is found, then the merge cannot be processed.

  3. Merge old Tissue into new Tissue

    Merges two distinct Tissue records into the New Tissue and deletes the Old Tissue. Propagates changes out to all cross-references to the Old Tissue:

    1. Molecular Segment Library (Source)

Retrieving Other Tissue Information

  1. To retrieve all Data Sets which cross-reference the current Tissue, click on Retrieve Info in the Data Sets table. This will list the MGI: Accession Number for each Molecular Segment or GXD Antigen which uses the current Tissue.
