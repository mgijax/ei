GXD EDITING INTERFACE: Design and Requirements - Genotypes

Introduction

An expression assay is defined as a group of expression-related facts reported in a single reference, having a single type (e.g. Western blot), measuring the expression of a single gene and using a single probe. Within one assay, expression may be measured among different animals (having different genotypes and/or ages) and within different tissues.

If an assay is a Gel Assay (e.g. Northern), the the assay record corresponds to the gel itself.

If an assay is an In Situ (e.g. Immunohistochemistry), then the assay record corresponds to a somewhat abstract grouping of the animals and results for a single probe.

If a Gel or In Situ assay uses multiple probes, then it is split into separate assays when it is entered into the database.

Each Assay contains a nucleotide probe preparation record or an antibody preparation record. A probe (nucleotide or antibody) used in an assay is "prepared" by attaching a radioactive label to it so that it can be detected, e.g. by autoradiography.

Deletions

A Genotype cannot be deleted if it is referenced by a:
  1. Speciman
  2. Gel Lane

Miscellaneous

  1. The only Genotype records returned by a Search are those Genotypes records which are cross-referenced by the currently selected Assay record in the Assay module (if there is one) and those Genotype records which are not cross-referenced by any Assay.

  2. When a Genotype record is Added to the database, the Genotype record is automatically added to the Genotype Clipboard.
