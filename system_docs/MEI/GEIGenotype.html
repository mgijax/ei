GXD EDITING INTERFACE: Design and Requirements - Genotypes

Introduction

The Genotypes module is used to define Strain and Mutant Allele pairs for InSitu and Gel assays.

Editing Form

  1. GIF Image

Data Verifications

  1. Strain (see Data Verifications section on Strain)

  2. Markers (see Data Verifications section on Markers)

  3. Marker Alleles (see Data Verifications section on Marker Alleles)

Fields

  • Required
    1. Strain
    2. If a Marker is specified, then both Allele 1 and Allele 2 are required.

Defaults

  1. Strain - Not Specified

Deletions

A Genotype cannot be deleted if it is referenced by a:
  1. Speciman
  2. Gel Lane

Miscellaneous

  1. The only Genotype records returned by a Search are those Genotypes records which are cross-referenced by the currently selected Assay record in the Assay module (if there is one) and those Genotype records which are not cross-referenced by any Assay.

  2. When a Genotype record is Added to the database, the Genotype record is automatically added to the Genotype Clipboard.
