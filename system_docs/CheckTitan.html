Check Titan Processes

The editing interface processes which run on the titan server must be monitored to detect obsolete processes. Obsolete editorial processes will be left running on titan when a user's MacIntosh crashes or the application is exited in such a way that the appropriate signal is not sent to the titan server process so that it can be killed.

Perform a ps -ef | grep run command on titan to get a listing of all currently running editorial interface processes: 

     plg  5832   287  0   Mar 13 ?        0:00 csh -c setenv DISPLAY 192.233.41.12:0.0; /usr/local/mgi/ei/mgd/runmgd

If any of the STIME columns (5th column) contains a date value which is less than today's date, then that process needs to be killed: 
	titan> su
	titan> ps -ef | grep plg
     		plg  5832   287  0   Mar 13 ?        0:00 csh -c setenv DISPLAY 192.233.41.12:0.0; /usr/local/mgi/ei/mgd/runmgd

     		plg  5833  5832  0   Mar 13 ?        0:21 mgd
